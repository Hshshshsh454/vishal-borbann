# 📸 Vishal Borban Photography Portfolio

A modern photography portfolio website built using **React + TypeScript**.  
This project showcases wedding photography services, portfolio gallery, pricing packages, and contact sections.

---

## 🚀 Features

- Responsive modern UI
- Hero landing section
- Wedding gallery grid
- Services showcase
- Pricing packages
- FAQ section
- Contact form
- Smooth layout and professional photography theme

---

## 🛠 Tech Stack

- React
- TypeScript
- CSS3
- Vite
